形容词+助动词

## [[い形容詞]]

| 简体  | 现在   | 过去     |
| --- | ---- | ------ |
| 肯定  | (い)  | かった    |
| 否定  | く　ない | く　なかった |

| 敬体  | 现在      | 过去        |
| --- | ------- | --------- |
| 肯定  | (い)　です  | かった　です    |
| 否定  | く　ない　です | く　なかった　です |

特殊：「いい」要变成「よい」再变形

## [[な形容詞]]

| 敬体  | 现在    | 过去      |
| --- | ----- | ------- |
| 肯定  | だ     | だった     |
| 否定  | では　ない | では　なかった |

| 敬体  | 现在       | 过去           |
| --- | -------- | ------------ |
| 肯定  | です       | でした          |
| 否定  | では　ありません | では　ありません　でした |
